# Public 1000 Genomes LD Window

This optional experiment uses public IGSR/1000 Genomes genotypes and the draft
`genotype_pair_ld` application to compute adjacent-pair genotype covariance and
LD-style `r2` summaries over a small chr22 window.

Run it from the repository root:

```bash
bash docs/paper/experiments/e8_public_ld_window.sh
```

The public paper evidence page is:

```text
https://blindmachine.org/verify/paper/public-genomics-e8-ld-window
```
